HOW IT WORKS - 5 STEP PROCESS:
================================================================================

STEP 1: Backend Gets Angel One Index Tokens
─────────────────────────────────────────────
  Backend defines in code:
    - NIFTY 50 token: "99926000" (NSE Exchange)
    - SENSEX token: "99919000" (BSE Exchange)

STEP 2: Backend Connects to Angel One WebSocket
───────────────────────────────────────────────
  - Establishes WebSocket connection to Angel One
  - Subscribes to these index tokens
  - Listens for real-time trade ticks

STEP 3: Every Trade Update → Backend Receives Tick
──────────────────────────────────────────────────
  When NIFTY 50 trades at 23,456.78:
    Angel One → WebSocket Tick → Backend handleTradeUpdate()
    
  When SENSEX trades at 75,432.10:
    Angel One → WebSocket Tick → Backend handleTradeUpdate()
    
  Backend stores in memory:
    currentPrices["NIFTY 50"] = 23,456.78  ← LIVE
    currentPrices["SENSEX"] = 75,432.10    ← LIVE

STEP 4: Backend Broadcasts to Frontend
──────────────────────────────────────
  Backend sends via Socket.io:
    io.emit("priceUpdate", {
      "NIFTY 50": 23,456.78,
      "SENSEX": 75,432.10,
      ... (all other stocks)
    })
    
  This happens:
    - Immediately when a tick arrives (real-time)
    - Every 2 seconds as fallback (reliability)

STEP 5: Dashboard Receives & Displays
──────────────────────────────────────
  Frontend TopBar.js Component:
    
    1. Receives Socket.io "priceUpdate" event
    2. Gets latest prices:
         currentPrice["NIFTY 50"] = 23,456.78 (from Angel One)
    3. Gets opening price (fetched from backend at startup):
         openingPrice["NIFTY 50"] = 23,000.45 (from Angel One)
    4. Calculates % change:
         changePercent = ((23,456.78 - 23,000.45) / 23,000.45) * 100
                       = +1.98%
    5. Determines color:
         If current > opening → GREEN (up)
         If current < opening → RED (down)
    6. Displays in top-left:
         NIFTY 50: 23,456.78
         +1.98% ✓ GREEN

UPDATE FREQUENCY:
================================================================================

DURING MARKET HOURS (9:15 AM - 3:30 PM IST):
  - WebSocket ticks: Hundreds of times per second (every trade)
  - Frontend updates: Real-time (100+ updates/sec on volatile days)
  - Status shows: "● MARKET LIVE"

AFTER MARKET HOURS:
  - WebSocket ticks: NONE (market closed)
  - Fallback timer: Every 2 seconds (simulated small movement)
  - Frontend updates: Every 2 seconds
  - Status shows: "● FEED LIVE (MARKET CLOSED)" or "● SIMULATED FEED"

DURING ANGEL ONE DISCONNECT:
  - WebSocket: Connection lost
  - Fallback timer: Still runs (simulates prices)
  - Frontend updates: Every 2 seconds
  - Status shows: "● SIMULATED FEED"
  - Reason: Ensures dashboard never freezes
